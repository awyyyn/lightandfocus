## Getting Started

#### First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

#### Building Project
```bash
npm run build
# or
yarn build
# or 
pnpm build
```

## Project Purpose

The project is to showcase the Light and Focus business to everyone. The user can save the package they want and also can send a message through the Message us section.
